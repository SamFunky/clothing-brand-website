# VANTA — Wear The Dark

A dark, edgy streetwear e-commerce site. Heavyweight tees, hoodies, crewnecks, and baggy denim.

## Stack

- Vite + React 19 + TypeScript
- Tailwind CSS v4
- Framer Motion (scroll/entrance animations)
- Lenis (smooth scrolling)

## Run

```bash
npm install
npm run dev
```

## Images

All campaign/product imagery lives in `public/images/` and is generated with Higgsfield:

| File | Purpose |
| --- | --- |
| `hero.jpg` | Full-screen hero (16:9) |
| `product-*.jpg` | Product cards (3:4) |
| `look-1.jpg` … `look-3.jpg` | Lookbook grid |
| `manifesto.jpg` | Manifesto section detail shot |

Missing images fall back to dark placeholders automatically.
